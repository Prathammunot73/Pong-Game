# 🏓 Pong Game

A classic Pong game built using Python and the Turtle graphics module as part of the **100 Days of Code: Python Bootcamp**.

## Features

- Two-player gameplay
- Ball collision detection
- Paddle movement
- Score tracking
- Increasing ball speed after each paddle hit

## Controls

### Right Player
- ↑ : Move Up
- ↓ : Move Down

### Left Player
- W : Move Up
- S : Move Down

## Tech Stack

- Python
- Turtle Graphics

## Project Structure

```
Pong-game/
│── ball.py
│── main.py
│── paddle.py
│── scoreboard.py
```

## How to Run

1. Clone this repository.
2. Install Python 3.
3. Run:

```bash
python main.py
```
